# ey-ai-finance-infra

Azure infrastructure for EY AI Finance POC environments. Bicep templates deploy a full per-POC stack: resource group, App Configuration, Key Vault, PostgreSQL (Citus), MongoDB (DocumentDB), Azure AI Search, Azure OpenAI, Storage, and frontend/backend App Services.

**Deploy:** See [bicep/README.md](bicep/README.md) for prerequisites, parameters, and deployment commands.
